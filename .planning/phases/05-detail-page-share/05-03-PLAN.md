<threat_model>
## Trust Boundaries

| Boundary | Description | Data Crossing |
| --- | --- | --- |
| Server page -> client share island | Server-derived share payload is handed to browser UI | share text, share-image URL |
| Client -> share-image route | Browser fetches same-origin PNG bytes for download | slug, PNG blob |

## STRIDE Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
| --- | --- | --- | --- | --- |
| T-05-07 | Tampering | copied share text | mitigate | Copy only normalized plain-text share content from the server helper; do not derive it ad hoc from rendered markdown. |
| T-05-08 | Information Disclosure | detail-page share UI | mitigate | Reuse published-only detail data and do not surface raw internal governance metadata directly in the client UI. |
| T-05-09 | Denial of Service | client download flow | mitigate | Keep same-origin download path simple and show inline failure guidance instead of repeated uncontrolled retries. |
</threat_model>

## PowerUp 是什么

PowerUp 是一个面向 **已发布 Skill** 与 **MCP Server** 的目录站。它不会试图把所有 AI 工具都塞进同一个入口，而是优先整理那些已经有公开资料、值得继续了解的条目，让你可以浏览、筛选、搜索，并在详情页快速判断它是否适合自己的工作流。

## 这张目录想解决什么

- 让你先看到值得关注的条目，而不是先理解复杂的提交流程或后台结构。
- 让首页就能完成搜索、筛选、排序和浏览，减少来回跳转。
- 让详情页提供足够的信息，帮助你判断一个 Skill 或 MCP 是否值得继续收藏、测试或接入。

## 收录范围

- 收录：MCP Server、Anthropic 风格 Skill。
- 不收录：GPTs / Agent 模板、Prompt 模板、OpenAI Plugin 风格 API 插件目录。
- 目前内容以 **编辑筛选** 为主，还不是开放投稿的平台。

## 目前的内容边界

- 目录主入口仍然是页面浏览，不是开放投稿或后台运营系统。
- 搜索以基础查找为主，先帮助你快速缩小范围，再进入详情做判断。
- 当前也提供只读 API 供前端消费，但不提供公开写接口。
- 页面会优先保证清晰阅读和直接浏览，让查找路径尽量简单。

## 这页不会解决什么

下面这些能力目前都不是这张目录的重点：

- FTS5 搜索索引、触发器同步和高亮片段。
- 即时搜索建议、防抖与键盘建议面板。
- 公开写接口、后台系统、投稿审核流、账号体系。
- `sitemap`、`robots`、JSON-LD。
- Docker / Caddy / 发布硬化。
- 深色模式、移动端抽屉导航和更复杂的前端状态管理。

## 联系与反馈

- 如果你发现信息需要更新，或希望推荐值得收录的条目，可以先通过项目仓库反馈。
- 目前仍以编辑筛选为主，暂不提供自助提交通道；后续若内容规模扩大，再补充更完整的投稿与审核方式。

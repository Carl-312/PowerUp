## PowerUp 是什么

PowerUp V1 是一个面向 **已发布 Skill** 与 **MCP Server** 的轻量目录站。它的目标不是把所有 AI 工具都塞进同一个入口，而是把已经完成基础核验、值得继续研究的条目整理成一个可浏览、可筛选、可搜索、可进入详情页判断接入价值的只读目录。

## V1 的目标

- 让用户能直接浏览已发布条目，而不是先理解复杂后台或内容工作流。
- 让首页承担完整结果态：筛选、排序、分页和 `/?q=` 搜索都回到首页。
- 让详情页帮助用户快速判断一个 Skill 或 MCP 是否值得继续接入、测试或收藏。

## 当前收录范围

- 收录：MCP Server、Anthropic 风格 Skill。
- 不收录：GPTs / Agent 模板、Prompt 模板、OpenAI Plugin 风格 API 插件目录。
- 当前内容是 **编辑精选目录**，不是开放自助提交平台。

## 当前边界

- 页面直接调用服务端查询层，不通过内部 HTTP API 绕行。
- 搜索先使用普通 SQLite 查询，不做 FTS5、触发器和高亮片段。
- 站点优先浅色、低 JavaScript、服务端渲染和基础可读性，不把复杂前端状态管理作为首版前提。

## 非首版阻塞项

以下能力明确不作为 V1 MVP 的上线阻塞项：

- FTS5 搜索索引、触发器同步和高亮片段。
- 即时搜索建议、防抖与键盘建议面板。
- 公开 API。
- `sitemap`、`robots`、JSON-LD。
- Docker / Caddy / 发布硬化。
- 深色模式、移动端抽屉导航和更复杂的前端状态管理。

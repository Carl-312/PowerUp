# PowerUp Specs V1

`specs-v1/` 只保留 V1 的产品与 contract 层文档。

## 当前角色

- `PRD.md`
  - 产品目标、范围、验收标准
- `terms-and-directory-contract.md`
  - 术语、目录职责、类型枚举、分类 slug contract
- `routing-render-search-contract.md`
  - 路由职责、渲染模式、搜索流转 contract
- `_history/`
  - walkthrough 类历史执行文档

## 历史边界

- `walkthrough.md` 与 `walkthroughV2.md`
  - 当前都只是根级跳转桩
- 真实历史正文在 `_history/walkthrough.md` 与 `_history/walkthroughV2.md`

## 使用边界

- 了解当前实现，先看 `../docs/**`
- 看运行命令，去 `../site/README.md`
- 看后续执行层，去 `../.planning/**`
- 不要把 `_history/**` 当成当前实现真源
